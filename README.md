# CAN Converter Unit
</div>

Zephyr RTOS firmware for the STM32G491CE-based RS485 to CAN converter.  
Receives telemetry from the master controller over RS485 (COBS-framed Protocol Buffers) and re-broadcasts it on a 1 Mbit/s CAN bus.

**PCB:** [HydrogreenPollub/stm32-can-converter-unit](https://github.com/HydrogreenPollub/stm32-can-converter-unit)
